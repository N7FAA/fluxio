# Fluxio 产品需求文档 · 页面与交互

> 基于 fluxio.pen 设计稿，定义 Fluxio 前端页面结构与交互规范

---

## 一、页面视图清单

| 视图 ID | 设计稿名称 | 说明 | 触发条件 |
|---------|------------|------|----------|
| `home-empty` | Fluxio-图片 / 动态图像 / 视频 | 首页空态，无文件 | 初始 / 清空后 |
| `home-dragging` | Fluxio-空态（拖拽中） | 拖拽文件悬停上传区 | 拖拽进入上传区 |
| `output-settings` | Fluxio-输出设置（图片/动图/视频） | 左面板文件网格 + 右面板输出设置 | 有文件时 |
| `file-hover` | Fluxio-输出设置 - 文件hover | 文件卡片悬停显示删除 | 鼠标悬停文件卡片 |
| `task-progress` | Fluxio-任务列表（进行中） | 任务进行中列表 | 有任务 running/queued |
| `task-completed` | Fluxio-任务列表（已完成） | 任务已完成列表 | 全部任务完成 |
| `view-file` | Fluxio-查看文件（图片/动图/视频） | 预览详情 + 信息面板 | 点击「查看」 |

## 二、页面结构与布局

### 2.1 首页空态（home-empty）

```
┌────────────────────────────────────────────────────────────┐
│  Fluxio, 让动效轻松落地                                      │
│  面向线上页面的开放转码工具                                  │
├────────────────────────────────────────────────────────────┤
│  [ 图片 ]  [ 动态图像 ]  [ 视频 ]     ← Tab                 │
├────────────────────────────────────────────────────────────┤
│  ┌──────────────────────────────────────────────────────┐  │
│  │  [上传图标]                                            │  │
│  │  支持点击或拖拽上传                                    │  │
│  │  *支持 PNG、JPG、WEBP 格式（按 Tab 变化）              │  │
│  │  [ 上传文件 ]                                          │  │
│  └──────────────────────────────────────────────────────┘  │
│                                                            │
│  输出设置（置灰/禁用）                                      │
│  格式 [WEBP ▼]  分辨率 [原始 ▼]  质量 [推荐 ▼]            │
│                                                            │
│  [ 开始转换 ]（禁用）                                       │
├────────────────────────────────────────────────────────────┤
│  转换完成后将在此显示                                        │
└────────────────────────────────────────────────────────────┘
```

**交互**：
- Tab 切换：更新上传区格式提示、输出设置选项
- 点击上传 / 拖拽：添加文件后切换至 `output-settings`
- 输出设置、开始转换：禁用

---

### 2.2 空态拖拽中（home-dragging）

**视觉变化**：
- 上传框：border 2px dashed #cbd5e1
- 上传图标：禁用/置灰
- 输出设置：禁用
- 开始转换：禁用

**交互**：松开拖拽 → 执行上传，成功后切换至 `output-settings`

---

### 2.3 输出设置（output-settings）

```
┌─────────────────────────────────────────────────────────────────────┐
│  [ 图片 ]  [ 动态图像 ]  [ 视频 ]                                    │
├──────────────────────────────────┬──────────────────────────────────┤
│  Left Panel (#F8FAFC)             │  Right Panel (#FFFFFF)           │
│  左侧圆角 24px                    │  宽度 360px，右侧圆角 24px        │
│  ┌────┬────┬────┐                 │  输出设置                         │
│  │ 80 │ 80 │ 80 │  文件卡片网格   │  格式：○ WEBP ○ JPG ○ PNG        │
│  │ ×80│ ×80│ ×80│  每卡 180×180   │  质量：体积优先 | 推荐 | 清晰优先  │
│  │文件名│文件名│文件名│             │  分辨率：○ 原始 ○ 0.75x ○ 0.5x   │
│  └────┴────┴────┘                 │  [ 开始转换 ]                     │
│  [ 清空 ]  [ 添加文件 ]           │                                  │
└──────────────────────────────────┴──────────────────────────────────┘
```

**文件卡片**：
- 80×80 预览区（图片/动图/视频缩略图或占位）
- 文件名（14px，#64748B）
- 悬停：右上角显示删除按钮（trash-2 图标）

**交互**：
- 清空：移除全部文件，返回 `home-empty`
- 添加文件：打开文件选择
- 删除（hover 显示）：移除该文件
- 开始转换：提交任务，切换至 `task-progress`

---

### 2.4 任务列表（task-progress / task-completed）

```
┌────────────────────────────────────────────────────────────┐
│  任务列表                    正在处理中（1/4）               │
├────────────────────────────────────────────────────────────┤
│  [✓] 文件1.webp  13.8M | 1920×1080    [查看] [下载]         │
│  [○] 文件2.webp  99% 处理中...                              │
│  [ ] 文件3.png  排队中...                                   │
├────────────────────────────────────────────────────────────┤
│  [ 回到首页 ]  [ 下载全部 ]                                  │
└────────────────────────────────────────────────────────────┘
```

**任务项状态**：
- 完成：绿色勾、大小、尺寸、查看/下载
- 进行中：进度环 + 百分比
- 排队：灰色占位

**交互**：
- 查看：打开 `view-file` 预览
- 下载：触发下载
- 回到首页：返回主流程（保留结果区）
- 下载全部：ZIP 打包下载（≥2 个成功时显示）

**交互**：
- 返回：关闭预览，回到任务列表
- 图片/动图：img 或 video loop 预览
- 视频：video 带 controls

## 三、状态流转

```
首页 (home) ──(上传文件)──> 输出设置页 (output-settings)
     │                            │
     │                            ├──(清空)──> 首页
     │                            │
     │                            └──(开始转换)──> 任务列表页 (task-list)
     │                                              │
     │                                              ├── 进行中 / 已完成
     │                                              ├── 查看、下载
     │                                              └──(回到首页)──> 首页
     │
     └──(拖拽进入)──> home-dragging ──(松开)──> 输出设置页
```

**页面说明**：
- **首页**：仅上传区，上传成功后跳转输出设置页
- **输出设置页**：文件网格 + 输出设置 + 开始转换，点击后跳转任务列表页
- **任务列表页**：展示进行中/已完成任务，可查看、下载，点击「回到首页」开始新一轮或关闭页面

**view-file**：从任务列表的「查看」进入，模态叠加
