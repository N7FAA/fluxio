# Fluxio 产品规格文档 Final

> 本文档合并 PRD、交互文档、补充决议、页面状态流转、设计稿映射，为 Fluxio 当前版本的唯一产品与实现依据。

---

# 第一部分：产品定位

## 1.1 产品概述

| 项目 | 内容 |
|------|------|
| **产品名** | Fluxio |
| **Slogan** | 让动效轻松落地 |
| **定位** | 面向线上页面落地的开放转码工具，帮助设计师将图片/动图/视频转为网页可用格式 |
| **三个 Tab** | 图片、动态图像、视频（按**目标输出类型**分流，非输入素材类型） |
| **核心能力** | 格式转换、分辨率缩放、质量档位、序列帧 ZIP 支持 |

## 1.2 目标与约束（当前版本）

| 项目 | 说明 |
|------|------|
| **目标** | 按本文档完成 Fluxio 页面，跑通「上传 → 转换 → 下载」全流程 |
| **范围** | 个人本地测试，不对外、不部署公网 |
| **成本** | 零：本机 Node + FFmpeg，无云服务 |
| **基础** | 复用现有 ffmpeg-tool 后端，新增 Fluxio 风格前端 |

## 1.3 文件与大小限制（PRD）

| Tab | 单文件上限 | 单任务文件数 | 单任务总大小 |
|-----|------------|--------------|--------------|
| 图片 | 30MB | 30 | 300MB |
| 动态图像 | 100MB(GIF/APNG/WEBP) / 500MB(MP4/WEBM/ZIP) / 1GB(MOV) | 10 | 2GB |
| 视频 | 同上 | 10 | 2GB |

**视频时长**：动态图像 Tab ≤20 秒，视频 Tab ≤60 秒（前端校验）

# 第二部分：实现口径（必须严格遵守）

| 项目 | 规则 |
|------|------|
| **视频时长校验** | 全部由前端完成，使用 HTML5 video metadata 读取 duration |
| **taskId** | 点击「开始转换」且校验通过后立即生成，格式：`task_${timestamp}_${random}` |
| **fileId** | 任务创建时按文件顺序生成，格式：`file_0`、`file_1`、`file_2`... |
| **sourceTab** | 任务创建时记录，使用 state + sessionStorage 存储 |
| **直接访问路由** | 若状态不存在，显示错误态，不做自动恢复 |
| **下载全部** | 全部完成后才可点击，当前版本采用逐个下载 |
| **回到首页** | 清空前端状态，保留任务来源 Tab |
| **Tab 切换确认** | 有文件时使用简单确认弹窗 |
| **失败态** | 仅做基础失败态，不做重试 |
| **文件详情** | 独立路由页面，**非弹窗**；时长/帧率按「有则显示，无则隐藏」 |

# 第四部分：路由与页面状态流转

## 4.1 路由

| 路由 | 对应页面 | 说明 |
|------|----------|------|
| `/` 或 `/fluxio` | 首页（P01~P07） | 空态与工作台为**局部状态切换**，不新增路由 |
| `/fluxio/tasks/:taskId` | 任务列表（P08/P09） | 进行中/已完成为同一路由状态切换 |
| `/fluxio/tasks/:taskId/files/:fileId` | 文件详情（P10/P11/P12） | 独立路由，**非弹窗** |

## 4.2 状态流转图

```
/ 或 /fluxio （首页）
├─ 空态 P01/P02/P03
│   ├─ 切换 Tab（无文件）→ 对应空态
│   ├─ 切换 Tab（有文件）→ 确认 → 清空 → 对应空态
│   ├─ 拖入 → P04
│   └─ 上传成功 → P05/P06/P07（局部切换，不跳路由）
├─ 拖拽中 P04
│   ├─ 拖离 → 当前空态
│   └─ 释放合法文件 → P05/P06/P07
└─ 工作台 P05/P06/P07
    ├─ 删除最后/清空 → 当前空态
    └─ 开始转换 → /fluxio/tasks/:taskId

/fluxio/tasks/:taskId
├─ P08 进行中 → 全部完成 → P09 已完成
├─ 查看 → /fluxio/tasks/:taskId/files/:fileId
└─ 回到首页 → /fluxio（保留 sourceTab）

/fluxio/tasks/:taskId/files/:fileId
└─ 返回 → /fluxio/tasks/:taskId
```

## 4.3 直接访问路由处理

**任务列表**：若找不到任务 → 错误态「当前任务不存在或已失效，请返回首页重新开始」+ 返回首页按钮

**文件详情**：若找不到任务或文件 → 错误态「当前文件不存在或已失效，请返回任务列表或重新开始」+ 返回任务列表/返回首页按钮

**不实现**：URL 直达后的任务恢复、刷新后的任务恢复、从后端重新拉取任务详情

# 第五部分：全局规则

## 5.1 Tab 规则

| 项目 | 规则 |
|------|------|
| Tab 含义 | 目标输出类型 |
| Tab 项 | 图片 / 动态图像 / 视频 |
| 默认选中 | 图片 |
| 切换 Tab（无文件） | 直接切换 |
| 切换 Tab（有文件） | 弹确认：「切换分类将清空当前已上传文件和设置，是否继续？」→ 确认后清空并切换 |
| 切换后 | 进入对应 Tab 空态 |

## 5.2 工作区规则（首页）

| 项目 | 规则 |
|------|------|
| 上传成功后 | 空态**替换**为工作台，不跳路由 |
| 删除最后 1 个文件 | 回到对应 Tab 空态 |
| 清空 | 回到对应 Tab 空态 |
| 添加文件 | 在当前工作台追加文件 |
| 处理中 | 不允许编辑，不允许返回工作台，必须通过「回到首页」开始新任务 |

## 5.3 sourceTab 存储

| 项目 | 规则 |
|------|------|
| 记录时机 | 点击「开始转换」创建任务时 |
| 取值 | `image` / `animated` / `video` |
| 存储 | state + sessionStorage（键如 `fluxio_task_${taskId}_source_tab`） |
| 回到首页 | 优先读取 sourceTab，返回对应空态 |
| 兜底 | 无法读取时默认 P01 |

# 第六部分：ID 生成规则

| 项目 | 规则 |
|------|------|
| **taskId** | 点击「开始转换」且校验通过后立即生成，格式 `task_${timestamp}_${random}` |
| **fileId** | 任务创建时按文件顺序生成，格式 `file_0`、`file_1`... |
| **路由示例** | `/fluxio/tasks/task_1720000000000_ab12cd/files/file_0` |

# 第七部分：视频时长校验

| Tab | 时长限制 |
|-----|----------|
| 动态图像 | ≤ 20 秒 |
| 视频 | ≤ 60 秒 |

**实现**：前端使用 `URL.createObjectURL` + 隐藏 `video` 元素 + `loadedmetadata` 读取 `duration`

**错误文案**：动态图像 Tab 超时长 / 视频 Tab 超时长 / 元数据读取失败（见第十一部分）

# 第九部分：页面级交互与按钮行为

## 9.1 首页空态（P01/P02/P03）

| 触发 | 条件 | 系统反馈 | 下一状态 |
|------|------|----------|----------|
| 点击上传 | 选择合法文件 | 解析并展示文件卡片 | P05/P06/P07 |
| 拖入合法文件 | 文件进入上传区 | 高亮上传框 | P04 |
| 拖入非法文件 | 格式不匹配 | 错误提示 | 当前空态 |

## 9.2 首页拖拽中（P04）

| 元素 | 状态 |
|------|------|
| 上传框边框 | 高亮 |
| 文案 | “释放以上传文件” |

| 触发 | 条件 | 系统反馈 | 下一状态 |
|------|------|----------|----------|
| 拖离 | 无 | 恢复默认 | P01/P02/P03 |
| 释放合法文件 | 当前 Tab 接受 | 执行上传 | P05/P06/P07 |
| 释放非法文件 | 当前 Tab 不接受 | 错误提示 | 当前空态 |

## 9.3 工作台（P05/P06/P07）

| 触发 | 条件 | 系统反馈 | 下一状态 |
|------|------|----------|----------|
| 点击删除 | 至少剩余 1 个 | 删除该文件 | 当前工作台 |
| 点击删除 | 删除后为空 | 清空工作区 | P01/P02/P03 |
| 点击清空 | 有文件 | 清空全部 | P01/P02/P03 |
| 点击添加文件 | 选择合法文件 | 追加到列表 | 当前工作台 |
| 点击开始转换 | 校验通过 | 生成 taskId，跳转 | P08 |

## 9.4 任务列表（P08/P09）

| 触发 | 条件 | 系统反馈 | 下一状态 |
|------|------|----------|----------|
| 点击查看 | 该文件已完成 | 跳转详情页（独立路由） | P10/P11/P12 |
| 点击下载 | 该文件已完成 | 下载单文件 | 当前 |
| 点击下载全部 | 全部完成 | 逐个下载 | 当前 |
| 点击下载全部 | 未全部完成 | 禁用 | - |
| 点击回到首页 | 任意时刻 | 清空状态，返回首页 | P01/P02/P03（保留 sourceTab） |

## 9.5 文件详情（P10/P11/P12）

| 触发 | 条件 | 系统反馈 | 下一状态 |
|------|------|----------|----------|
| 点击返回 | 无 | 返回任务列表 | P08/P09 |

**文件信息字段**：文件名、格式、尺寸、大小；动图/视频显示时长；动图显示帧率。时长/帧率按「有则显示，无则隐藏」。

# 第十一部分：异常与提示文案

| 场景 | 文案 |
|------|------|
| 当前 Tab 不支持该格式 | 当前分类不支持该文件格式，请切换到对应分类后重试 |
| 图片文件过大 | 图片文件不能超过 30MB |
| 动图文件过大 | 动态图像文件不能超过 100MB |
| MP4/WEBM 过大 | 视频文件不能超过 500MB |
| MOV 过大 | MOV 文件不能超过 1GB |
| ZIP 过大 | 序列帧 ZIP 不能超过 500MB |
| 动图 Tab 视频超时长 | 动态图像输出仅支持 20 秒以内的视频 |
| 视频 Tab 视频超时长 | 当前仅支持 60 秒以内的视频 |
| 元数据读取失败 | 无法读取视频信息，请更换文件后重试 |
| 上传失败 | 上传失败，请重试 |
| 转换失败 | 转换失败，请重试 |
| 任务不存在 | 当前任务不存在或已失效，请返回首页重新开始 |
| 文件不存在 | 当前文件不存在或已失效，请返回任务列表或重新开始 |

# 第十二部分：技术方案

## 12.1 后端（现有 server.js）

| 能力 | 接口 | 说明 |
|------|------|------|
| 上传 | POST /api/upload | 返回 `{ path, name, size }` |
| 序列帧 | POST /api/upload-sequence | 返回 `{ path, name, frameCount, pattern, width, height }` |
| 转码 | POST /api/queue | 接收 inputPath/inputSequence、container、params |
| 任务状态 | GET /api/jobs | 含 outputPath、status、progress、outputSize |
| 下载 | /uploads/文件名 | 静态文件 |
| 下载全部 | GET /api/download-zip?ids=... | 当前版本前端改用逐个下载 |

## 12.2 前端路由实现

- 使用 History API 或 Hash 路由
- 服务端需支持：`/fluxio`、`/fluxio/tasks/:taskId`、`/fluxio/tasks/:taskId/files/:fileId` 均返回 fluxio.html

## 12.3 参数映射（queue payload）

| Tab | 格式 | 映射 |
|-----|------|------|
| 图片 | webp/png/jpg | container |
| 动态图像 | webp/apng/gif | container + frameRate |
| 视频 | mp4/webm | container + frameRate |
| 通用 | 分辨率 | params.scale |
| 通用 | 质量 | params.crf (30/35/23) |
